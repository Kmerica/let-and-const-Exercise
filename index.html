Let and Const Exercise

***Quiz***

  • What is the difference between var and let?

    • A variable declared using let is block-scoped and only visible within the block in which it was declared. It may be reassigned but not redeclared. While a variable declared using var may be reassigned or redeclared.

  • What is the difference between var and const?

    • Var may be reassigned or redeclared, but when using const a variable may not be redeclared or reassigned. A variable declared using the keyword const is also block-scoped.

  • What is the difference between let and const?

    • When a variable is declared with const it may not be reassigned or redeclared. When a variable is declared with let it may be reassigned.

  • What is hoisting?

    • When a variable is moved to the top of its scope before code execution. A variable declared using var can be accessed before it is defined in the codebase.
